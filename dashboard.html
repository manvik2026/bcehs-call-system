Quick Setup

Step-by-step deployment guide
1

Install dependencies

pip install -r requirements.txt

2

Configure environment

Copy .env.example → .env and fill in Twilio + Anthropic keys

3

Run Flask server

python app.py — starts on port 5000

4

Expose via ngrok

ngrok http 5000 — copy the HTTPS URL

5

Set Twilio webhook

In Twilio Console → Phone Numbers → set Voice URL to https://your-ngrok.app/voice

6

Call & test!

Dial the Twilio number. Or run python test_call.py for local simulation.
